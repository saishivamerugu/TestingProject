# Git & GitHub Learning Repository

This repository is created for learning and mastering **Git** and **GitHub**.

## About This Repository

Here I will practice:
- Git basics and commands
- GitHub repository management
- Branching and merging
- Commit history and version control workflows
- Pull requests and collaboration
- Resolving conflicts
- Working with remote repositories

## Purpose

The main purpose of this repo is to:
- Learn Git step by step.
- Understand how GitHub works.
- Track my progress while practicing version control.
- Build confidence in using Git for real projects.

## Topics Covered

- `git init`
- `git status`
- `git add`
- `git commit`
- `git log`
- `git branch`
- `git checkout` / `git switch`
- `git merge`
- `git remote`
- `git push`
- `git pull`

Used this repository as a practice space for:

4. Testing merges and conflict resolution.
